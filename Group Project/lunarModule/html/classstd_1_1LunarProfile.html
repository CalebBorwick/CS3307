Detailed Description

The LunarProfile class: class containing all functions nessisary for the LunarProfile.

The LunarProfile class contains the getters and setters for all parameters as well as a constructor and destructor for making and destorying objects of type LunarProfile as well as the apicall and QtNework information for getting and requesting information from an api. The class also contains the function to create a gui to display the lunar phase information recieved from the api call
