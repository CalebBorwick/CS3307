Detailed Description

The clockProfile class: class contains all functions necessary for the clockProfile.

The clockProfile class conbtains the getters and setters for all parameters as welll as a constructor and destructor for make and destroying objects of type clockProfile. As well it holds the functions to create and diosplay a gui with the all information as well as the ability to change the deafults using getUserInput()
