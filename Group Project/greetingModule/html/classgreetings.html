Detailed Description

The greetings class: class containing all functions nessisary for the greetings.

The greetings class contains a constructor and destructor for making and destorying objects of type greetings as well as the apicall and QtNework information for getting and requesting information from an api. The class also contains the function to create a gui to display the greetings and motivational quote information recieved from the api call
